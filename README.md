The Food Delivery Web Application is a dynamic and responsive full-stack web application developed using Java technologies.
This project allows users to browse restaurants, view menus, place orders, and manage their profiles efficiently through an interactive and user-friendly interface.

It also includes an Admin Dashboard that helps administrators manage users, restaurants, orders, and analytics in real-time.

This application follows the MVC Architecture and is developed using Servlets, JSP, JDBC, and MySQL to ensure smooth data interaction between the frontend and backend.
